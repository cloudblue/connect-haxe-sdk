interface ILoggerFormatter

package connect.logger

Represents a log formatter.The Logger uses an instance of a class that implements this interface (PlainLoggerFormatter or MarkdownLoggerFormatter by default, depending on the value of LoggerConfig.markdown()) to write log messages.
